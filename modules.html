Modules



Modules are bundles of ready-made code that help you deal with common
coding challenges. Modules provide the exciting bits of code, letting you focus
on the fun stuff. Also, because modules are used by a lot of people, they are likely
to work well and be free of bugs. We use a module like import moduleName

datetime

This module lets you work
with dates. You can get
today's date, or work
out how long is until
a special day.

statistics

This module lets you to calculate averages
or find the most common value in a list
of numbers. It's handy if you need to
work out an average score in a game.

webbrowser

You can control the computer's web browser with this module, allowing you to open web page directly from code.

random

You can use this module to make random selections like in a Password Picker. It's great for adding an element of chance to your game or program.

socket

The socket module allows programs to communicate across internet and networks. It could be used to create a multiplayer game.
